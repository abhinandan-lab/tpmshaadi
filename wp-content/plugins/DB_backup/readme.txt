A plugin to download the WordPress database in one click.

== Description ==

DB Export is a simple and efficient plugin that allows you to download your WordPress database with a single click. It's designed to be user-friendly and provides a quick way to backup your database for safekeeping or migration purposes.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/db-export` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the 'DB Export' menu item in the WordPress admin to access the export functionality.

== Frequently Asked Questions ==

= How do I use the plugin to export my database? =

Simply navigate to the 'DB Export' menu item in your WordPress admin dashboard and click the 'Export Database' button.

= Is it safe to use this plugin? =

Yes, DB Export performs a read-only operation on your database to generate a backup file. It does not modify any data.

= Can I schedule automatic backups with this plugin? =

No, DB Export is designed for manual, on-demand exports only. For scheduled backups, consider using a comprehensive backup plugin.
